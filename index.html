Automated Bibliography Server for Research Publications Diffusion

Presentation

BasilicPlus is a bibliography server for research laboratories. It automates and facilitates the diffusion of research publications over the internet, automatically generating web pages from a publication database. Each publication has an associated web page, which provides downloads and additional documents (abstract, images, BibTeX...). Index pages are also created, including a search engine with several options for results display. New publications can be added to the database in an instant.

Download and configuration

Follow the instructions given on the install.html page to install BasilicPlus on your server.

The installation process is simple and entirely configurable. Your server's configuration is fully tested before installing anything.

The BasilicPlus look and feel is defined using CSS style sheets and can hence easily be customized. The first Basilic software was originally written for a Computer Graphics laboratory, which explains the thumbnails displayed in front of each publication. In BasilicPlus this feature has been removed.

Features

  • Main features
    • Automated bibliography server: easy to use, automatic page generation.
    • Online upload module to add files to publication with possibility to protect access
    • Each publications' web page automatically shows all of the associated documents.
    • Index pages with a powerful search engine.
    • Various result displays : researcher's publication list, BibTeX...
    • Various export types : BibTeX, Full Text...
    • Separate back-office for edition of publications and database checking.
    • Team end Project handling.
    • Personalize your publications list with a publication cart
  • Ease of use
    • New publications can be added and edited easily using a simple web form.
    • To publish associated documents (pdf files, images, abstract, ...), simply use the dedicated web form.
    • Thumbnails are generated automatically from images and movies.
  • Up-to-date web server
    • Publication lists and associated pages are always up to date.
    • Dynamic content: changes are immediately published.
    • Easy use encourages researchers to keep the server updated.
  • Simplified administration
    • Lightweight server administration: an e-mail is sent in case of problems.
    • Delegated management: each researcher manages his or her own publications.
    • Simple semi-automatic system installation
    • Initial database can be created from BibTeX files.
  • Clean organization
    • Clean directory hierarchy: sorted by year, then by publication BibTeX key.
    • Each publication's associated directory holds all of the associated documents.
    • A separate back-office section for bibliography management.
    • A unified look and feel for all publications.
    • Bilingual (or more) web sites are handled.
  • Conformity to standards
    • Validity of BibTeX entries is enforced.
    • Strict XHTML 1.0 and CSS 2.0 compliant pages are generated.
  • Customization
    • Customizable look and feel using css style sheets.
    • Distributed under the terms of the GNU-GPL licence and hence open source.
    • Based on simple php scripts that can easily be modified and extended.
    • Fully customizable installation paths to fit your web server organization.
  • Possible extensions
    • Adding new languages for multilingual web sites is easy.
    • Based on a mySQL database: new queries and tables are possible.
More technical details are provided in the BasilicPlus user's guide page.
